# storybook + vue3 + webpack 5

## 環境

- node: 14.19.1
- yarn: 1.22.18 `npm i -g yarn`
- npm/npx: 6.14.16
- @vue/cli: 5.0.8 `yarn global add @vue/cli`

## install

#### 1. @vue/cliをインストール

- `yarn create {プロジェクト名}`
- `yarn serve`

#### 2. Vue3の場合 webpack5を利用する。

- `npx sb init --builder webpack5`
- `yarn storybook`



## Project setup
```
yarn install
```
